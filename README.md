# Prompthub 
A curated gallery of powerful AI prompts designed to transform photos using tools like Gemini. It is a modern web platform for managing, exploring, and generating prompts efficiently. Explore cinematic, realistic, and artistic styles, copy prompts instantly, and create stunning edits with ease.


---

# Features

- Organized prompt storage
- Fast and responsive UI


## 🛠 Tech Stack

- HTML, CSS, JavaScript
- Hosting: Cloudflare
